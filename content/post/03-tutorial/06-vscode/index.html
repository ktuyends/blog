Visual Studio Code (VSCode) là một trong những trình soạn thảo mã nguồn phổ biến nhất được sử dụng bởi các lập trình viên. Nó nhanh, nhẹ, hỗ trợ đa nền tảng và gồm rất nhiều tính năng thú vị.

Trong bài viết này, tôi sẽ đi thực hiện từng bước để thiết kế một môi trường hỗ trợ cho công việc lập trình R của mình. Vì, ngôn ngữ chính tôi đang học tập là R. Bắt đầu nào!

1. Introduction to VSCode

Đầu tiên, các bạn vào các link dưới đây, Download phiên bản tương ứng với hệ điều hành và tiến hành cài đặt: VSCode

Sau khi cài đặt xong, các bạn bật VSCode lên, sẽ có giao diện giống như hình bên dưới:

1.2. Làm quen với giao diện của VSCode

Thanh sidebar:

Vùng giao diện này gồm 5 tính năng chính:

  • Explorer: Quản lý các file và thư mục đang làm việc.
  • Search: Tìm kiếm và thay thế.
  • Source Control: Quản lý các repository, version khi thực hành code.
  • Run: Thực thi code và debug.
  • Extensions: Quản lý các tiện ích mở rộng.

Thanh trạng thái:

Góc dưới bên trái của thanh trạng thái thể hiện các tình trạng về Git, số error và warning.

Góc dưới bên phải của thanh trạng thái thể hiện số dòng, số cột, định dạng encoding, ký tự kết thúc dòng (Có thể là LF hoặc CRLF) và cuối cùng là ngôn ngữ sử dụng.

Khung nhập lệnh:

Để bật khung nhập lệnh, sử dụng tổ hợp phím tắt: Ctrl + P

Khung nhập lệnh trên giúp ta tìm kiếm và mở các file nhanh hơn. Để chạy một lệnh nào đó cho VSCode, ta dùng tổ hợp phím: Ctrl + Shift + P. Để ý khung hiện ra tương tự khung trên, nhưng có thêm ký tự >

Ngoài các khung trên, VSCode còn có phần Menu Bar, Editors Tab, Panel và Minimap. Cái này khá dễ nhận biết, nên mình sẽ không đi thêm nữa.

1.3. Quản lý files và folders

Bước đầu tiên, khi làm việc với code chúng ta nên tạo ra các folders cho từng project. Để mở một folders hoặc project trong VSCode. Ta vào File -> Open Folder. Khi đó, toàn bộ các file trong folders sẽ được import vào trong VSCode.

Duyệt file và tìm kiếm: Cách đơn giản nhất là tìm các file bên trong thanh sidebar, hoặc dùng phím tắt: Ctrl + P và gõ tên file cần tìm kiếm vào trong khung nhập lệnh.

Tạo file và folder: Ta cũng có 2 cách, cách thứ nhất là chuột phải vào folders sau đó chọn tạo New file hoặc New folder. Cách thứ hai là sử dụng các icon trên thanh sidebar.

Với một số thao tác trực tiếp với files và folders, các bạn chuột phải vào file hoặc folder cần tương tác và lựa chọn hành động tương ứng.

1.4. Cài đặt và quản lý extension

Extensions hiểu đơn giản nó giống như các ứng dụng trên CH Play hoặc App Store. Chỉ khác là các ứng dụng này nhằm mở rộng các tính năng cho VSCode thôi.

Để tải và cài đặt một extension, trên thanh sidebar, bấm chọn Extension. Sau đó, tìm kiếm các Extension muốn cài đặt và chọn Install.

1.5. Multi selection

Phần này mình note lại là chính, nếu các bạn muốn tìm hiểu sâu về các tips trong VSCode thì nên đọc bài viết này: Visual Studio Code Tips and Tricks.

Multi cursor selection:

Đây là tính năng giúp các bạn lựa chọn và edit từ nhiều vị trí trong files code. Cơ bản nhất, để thêm con trỏ vào một vị trí tùy ý nào đó hãy sử dụng tổ hợp: Alt + Click

Tổ hợp phím: Ctrl + Alt + Up/Down - Đặt con trỏ ở cùng vị trí trên dòng trên hoặc dưới. Trường hợp một dòng trắng, sẽ đặt con trỏ vào đầu dòng.

Trường hợp, muốn thêm con trỏ cho tất cả các đoạn, giống đoạn đang được lựa chọn (bôi đen) sử dụng: Ctrl + Shift + L

Bỗng dưng, bạn chỉ muốn thêm con trỏ cho đoạn selection tiếp theo thay vì tất cả, sử dụng tổ hợp: Ctrl + D

Column (box) selection:

Lựa chọn văn bản theo khối, sử dụng tổ hợp: Shift + Alt + Kéo chuột
